¿Qué es un middleware?
Es un software que sirve para la comunicacion entre dos aplicaciones de software independientes.

¿Cuál es el significado del término “MEAN stack” ?
Es el conjunto de tecnologias utilizadas para desarrollar aplicaciones web en todos los niveles de full-stack.
Las cuales son MongoDB, ExpressJs, AngularJs y NodeJs

¿Cuáles son las características/funcionalidades que Express ofrece como middleware?
Integracion con motores de renderizacion de "vistas" para generar respuestas mediante la introduccion de datos en plantillas.
Establecer ajustes de aplicaciones web como que puerto usar para conectar, y la localizacion de las plantillas que se utilizan
para renderizar la respuesta.
Añadir procesamiento de peticiones "middleware" adicional en cualquier punto dentro de la tuberia de manejo de la petición.
Escritura de manejadores de peticiones con diferentes verbos HTTP en diferentes caminos URL.

¿Qué es una ruta en express?
Es la forma de declarar URL para la aplicación web

¿Qué es un “route handler” en express y para qué se utiliza?
Un route handler es la acción especifica que se toma cuando se entra a una ruta especifica.

¿Cuáles son los 2 parámetros necesarios para crear una ruta para el método GET con express, ejemplo: app.get()?
Necesitas la ruta en la cual se va a ejecutar  y la funcion con los parametros requisite y response 

¿Qué es lo que hace la instrucción res.send?
Envia la respuesta del HTTP

¿Cuál es el motivo por el que express es tan popular?
Es un framework de node muy facil de usar, y ofrece buen rendimiento, tiene soporte JavaScript y ayuda a construir las aplicaciones
en el lado del servidor de una manera mucho mas organizada con la arquitectura MVC
